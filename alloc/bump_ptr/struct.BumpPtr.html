pub struct BumpPtr { /* fields omitted */ }

A simple bump pointer allocator.

This allocator has few "moving parts" and is very fast. However, it doesn't allow objects to be deallocated. We use this allocator for early kernel objects before we can set up a more sophisticated heap allocator.
